About This Todo App

A personal todo list application built with Python Flask and SQLite.

Features

  • Add, edit, and delete todos
  • Toggle completion status
  • Responsive design for all devices
  • SQLite database for data persistence
  • Keyboard shortcuts (Escape to cancel, Ctrl/Cmd+Enter to add)
  • Inline editing with click to edit

Technology Stack

Backend
  • Python Flask
  • SQLite3 Database
Frontend
  • Bootstrap 5
  • Bootstrap Icons
  • Responsive CSS
  • Vanilla JavaScript
